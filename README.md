Inventory Management System

A console-based Inventory Management System developed using Java, MySQL, JDBC, and DAO Architecture. The project helps manage products, categories, suppliers, inventory stock, purchase orders, sales orders, billing, and user orders efficiently through a structured backend system.

🚀 Features

User Authentication (Admin & User Login)
Product Management (Add, Update, Delete, View)
Category Management
Supplier Management
Purchase & Sales Order Handling
Inventory Stock Tracking
Product Search & Filtering
Product Request System
Order History Management
Bill Generation System
OTP-Based Payment Verification
MySQL Database Integration
DAO Architecture Implementation
Console-Based User Interface

🛠️ Technologies Used

Java
MySQL
JDBC
Object-Oriented Programming (OOP)
DAO Design Pattern
XAMPP Server
